# 🩺 VetDordle

Jogo educativo de diagnóstico veterinário no estilo Doctordle. Casos clínicos são revelados progressivamente e o jogador deve adivinhar o diagnóstico usando o menor número de pistas possível.

## Como jogar

- Cada caso possui **6 pistas** progressivas (sinalamento, queixa principal, anamnese, exame físico, exames laboratoriais, exame complementar)
- Você tem **3 tentativas** para acertar o diagnóstico
- Quanto menos pistas revelar, mais pontos ganha
- O autocomplete ajuda com diagnósticos comuns

## Espécies cobertas

🐶 Cães · 🐱 Gatos · 🐴 Equinos · 🐄 Bovinos

## Casos incluídos

- Parvovirose Canina
- Cinomose
- Torção Gástrica (DVG)
- Erliquiose Canina
- Lipidose Hepática Felina
- Laminite Equina
- Obstrução Uretral Felina
- Babesiose Bovina

## Stack

- React (JSX)
- CSS-in-JS
- Sem dependências externas
